# Q87. “三航杯”是什么？
“三航杯”指西北工业大学“三航杯”大学生课外学术科技作品竞赛，同时也是“挑战杯”国赛的校内选拔赛。

（以下全为烧哥的讲解，太专业了QAQ）

> **烧锝氘锂**：“挑战杯”是由共青团中央、中国科协、教育部和全国学联共同主办的全国性的大学生课外学术实践竞赛，在大学生创新创业竞赛中处于全国顶尖的地位，与“互联网+”同为最高级别的大学生创新创业竞赛。“挑战杯”竞赛在中国共有两个并列项目，一个是“挑战杯”中国大学生创业计划竞赛（简称“小挑”）；另一个则是“挑战杯”全国大学生课外学术科技作品竞赛（简称“大挑”）。这两个项目的全国竞赛交叉轮流开展，每个项目每两年举办一届。“三航杯”与“挑战杯”分类相同，也分为两个赛道：创新赛道和创业赛道，但是不同的是，“三航杯”每年都会同时开办两个赛道的比赛。
> - 创新赛道：创新赛道之所以被称为“大挑”，是因为参赛项目和作品主要为：科技发明制作，自然科学类，哲学社会科学类社会调查报告和学术论文三种。
> - 创业赛道：创业计划作品竞赛以商业计划书评审、现场路演等作为参赛项目的主要评价内容，参赛者的参赛作品应是针对一项发明创造、技术专利或服务创新；参赛团队应在广泛进行市场调研、认真进行企业分析的基础上，完成一份把产品或服务推向市场的完整、具体、有实施可能的创业计划书。（烧从文件里复制来的，烧参加的是创新赛道，对创业不是很了解）项目根据不同领域分为不同组别。
> - 人员组成：参赛人员必须为西工大学生，本科生，研究生，博士生均可参赛（不含在职研究生）其中博士研究生不可作为负责人，且同一作品中博士研究生占比不超过 30%，作品贡献率不超过60%；但是作品贡献率较难评判，可以忽略不计。参赛人数上限10人，指导老师上限5人。
> - 获奖概率：“三航杯”初赛评选 60%左右的作品入围决赛，决赛设特等奖、一等奖、二等奖、三等奖，分别约占入围决赛作品总数的 3%、8%、24%和 65%。这是文件上说的，但是实际上，22年三航杯获奖程序为：初赛——三等奖和入围决赛名单公布——决赛。如果两个名单均未出现A项目，则A项目不获奖；如果A项目出现在三等奖名单中，则即为三等奖，不入围决赛，如果A项目出现在决赛名单中，则保底校赛二等奖，最高特等奖。
> - 比赛流程：各项目（各队）依据当年通知，大约在秋季学期11月底，注册“三航杯”平台，进行竞赛报名——赛道选择——作品提交，大约在12月初——资格审查（如果未通过，需进行修改，重新提交作品）——通过后进行竞赛初赛，由专家进行线上盲审，即对你在平台上提交的各项材料进行评分——初赛结果公布，会公布决赛名单和三等奖名单，大约在3月份——竞赛决赛，由答辩和路演两个环节组成，答辩为分组答辩，每组三个老师，路演在翱翔学生中心，每个项目一个海报一张桌子，然后会有老师和学生观众来看，就讲解就行了，大约在3月底——最终成绩公布，4月初。
> 公布结果后有些项目被选取参加“挑战杯”国赛，一般为“三航杯”一等奖或特等奖。烧是个二等奖，论文写烂了，所以也就不知道挑战杯是怎么一回事了。
